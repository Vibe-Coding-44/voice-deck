voice-deck

声がスライドの一次ソース

なにができる？

  • マイクで話すだけで Reveal.js スライドが生成される
  • 音声 → Whisper STT → Claude API → HTML の自動パイプライン
  • CLI 完結・ローカルファイル出力

コマンド一覧

voice-deck record          # マイクで録音してスライド生成
voice-deck build slide.md  # 既存 Markdown から HTML 生成
voice-deck preview out.html  # ローカルプレビュー
voice-deck from-wav audio.wav   # 保存済み音声から再開
voice-deck from-transcript stem # 文字起こしから再開
voice-deck artifacts        # 保存済みファイル一覧

エラー対策

  • 録音音声を ~/.voice-deck/artifacts/ に自動保存
  • Whisper / Claude API 失敗時も中間成果物で再開可能
  • RateLimit は最大 2 回リトライ（5 秒バックオフ）
  • クレジット不足時はダッシュボード URL を案内

アーキテクチャ

🎙 マイク
  ↓ sounddevice
🔉 WAV (自動保存)
  ↓ OpenAI Whisper API
📝 文字起こし (自動保存)
  ↓ Claude API (claude-sonnet-4-6)
📄 Markdown スライド
  ↓ build.py
🌐 Reveal.js HTML

インストール

git clone https://github.com/Vibe-Coding-44/voice-deck
cd voice-deck
python -m venv .venv && source .venv/bin/activate
pip install -e .
export OPENAI_API_KEY=sk-...
export ANTHROPIC_API_KEY=sk-ant-...
voice-deck record --title "My Talk"

まとめ

  • 声 = 一次ソース — Markdown を書く手間をゼロに
  • Git 管理できる .md ファイルとして出力
  • API 障害時も音声・テキストが残るフェイルセーフ設計
